class pysap.plugins.mri.parallel_mri.gradient.Grad2D_pMRI(data, fourier_op, S, linear_op=None, check_lips=False)[source]¶

Gradient for 2D parallel imaging reconstruction.

This class defines the datafidelity terms methods that will be defined by derived gradient classes: It computes the gradient of the following equation for the analysis and synthesis cases respectively:

  • (1/2) * sum(||Ft Sl x - yl||^2_2,l)
  • (1/2) * sum(||Ft Sl L* alpha - yl||^2_2,l)
