PySphinxDoc is a Python module for interactive sparse astronomical and MRI data analysis that offers:

  • a common API for astronomical and neuroimaging datasets.
  • an accces to 'sparse2d' using a wrapping or a binding strategy.
  • a user graphical interface to play with the provided functions.

piSAP¶

piSAP is a Python package related to sparsity and its application in astronomical or mediacal data analysis. This package binds the ‘sparse2d’ C++ library that allows sparse decomposition, denoising and deconvolution.
